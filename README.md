# 🛡️ CyberGuard Bot - Cybersecurity Awareness Chatbot

## Complete Solution (Parts 1, 2, and 3)

A comprehensive cybersecurity awareness application that educates users about online safety through an interactive chatbot with voice greetings, sentiment detection, task management, quiz, NLP, and activity logging.

## ✨ Features

### Part 1 - Console Chatbot
| Feature | Description |
|---------|-------------|
| 🎙️ Voice Greeting | Plays a recorded WAV file on startup |
| 🎨 ASCII Art Logo | Displays a cybersecurity-themed logo |
| 💬 Personalized Interaction | Asks for and remembers the user's name |
| 🔐 Cybersecurity Responses | Answers questions about passwords, phishing, privacy, and scams |
| ✅ Input Validation | Handles empty and invalid inputs gracefully |
| ⌨️ Typing Effect | Simulates natural conversation flow |
| 🎨 Colored Console UI | Uses colors and borders for visual appeal |

### Part 2 - WPF GUI Chatbot
| Feature | Description |
|---------|-------------|
| 🖥️ Modern GUI Interface | WPF with beautiful colors, chat bubbles, and styling |
| 🎯 Keyword Recognition | Uses **Dictionary<T>** for keyword-response mapping |
| 🎲 Random Responses | Uses **List<T>** for random response selection |
| 😊 Sentiment Detection | Detects worried, curious, frustrated moods |
| 🧠 Memory & Recall | Remembers user's name and favorite topics |
| 🔄 Conversation Flow | Handles "tell me more" and "another tip" commands |
| 💬 Chat Bubble Layout | Bot on left, user on right |

### Part 3 - Advanced Features
| Feature | Description |
|---------|-------------|
| 📋 Task Assistant | Add, view, complete, and delete cybersecurity tasks |
| 💾 Database Integration | SQLite database storage using **Entity Framework Core** |
| 🎯 Cybersecurity Quiz | **12 questions** with immediate feedback and score tracking |
| 🧠 NLP Simulation | Detects intents like "add task", "remind me", "start quiz" |
| 📝 Activity Log | Tracks all actions with "show activity log" command |
| 🔄 Show More | View more log entries with "show more" command |
