## Part 2 - WPF GUI Application

A WPF desktop application that educates users about cybersecurity.

---

## Features

- 🎙️ Voice greeting on startup
- 🎨 ASCII art logo display
- 💬 Personalized name memory
- 🔐 Keyword recognition (passwords, phishing, privacy, scams)
- 🎲 Random responses using Lists
- 😊 Sentiment detection (worried, curious, frustrated)
- 🔄 Follow-up handling ("tell me more")
- 🧠 Memory recall ("remember me")

---

## How to Run

1. Clone the repository
2. Open in Visual Studio
3. Press F5 to run
